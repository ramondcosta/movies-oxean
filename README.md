# movies-oxean

## Requirements
- [redis](https://redis.io/docs/install/install-redis/)
- ruby-3.1.4
- sqlite3

## Setup
    bundle install
    rails db:migrate
    rails db:seed

## execution steps
    rails s
    bundle exec sidekiq
